• DEWA (Dealer Web Gateway) Tim Proyek – proyek pengembangan portal internal Astra International berbasis Liferay DXP 7.0.11.
  • Mengembangkan antarmuka pengguna portlet Angular menggunakan Angular npm-portlet (Blade CLI) dengan Bootstrap dan SCSS untuk styling yang terstruktur, reusable, dan konsisten.
  • Membangun Spring Boot API untuk modul laporan, pelatihan, dan approval yang terintegrasi dengan portlet angular di dalam Liferay.
  • Mengembangkan logika backend menggunakan Hibernate dan JPA Repository untuk mengelola data transaksi dan dokumen.
  • Menerapkan mekanisme deployment dan debugging OSGi module di lingkungan server Astra untuk memastikan integrasi frontend–backend berjalan stabil.
