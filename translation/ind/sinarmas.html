• Onboarding-agent - Tim Proyek - Sebuah project yang fokus dalam perekrutan agent untuk penjualan produk asuransi sinarmas dengan sistem MLM
  • Mensupport proses calon agent ke agent yang sudah di approved berdasarkan informasi data di sinarmas database
  • Membangun backend menggunakan Spring Boot
  • Membangun query data kompleks di Oracle
  • Membangun integrasi API untuk frontend
  • Membangun backend untuk pemetaan data dari query database oracle ke kelas Java menggunakan framework Hibernate dengan Membangun JPA melalui JPA Repository
