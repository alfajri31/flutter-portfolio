• Onboarding-agent Project Team - A project focused on recruiting agents for the sale of Sinarmas insurance products using a Multi-Level Marketing (MLM) system.
  • Assisting in the process of converting potential agents to approved agents based on data information available in the sinarmas database
  • Developed with team backend using spring boot
  • Developed with team complex query data in oracle
  • Developed with team Integration API for frontend
  • Backend implementation for data mapping from Oracle database queries to Java classes using the Hibernate framework with JPA implementation through JPA Repository
